- **regular expression**
    - repetition
        - **" +"** matches 1 or more spaces in a row, if find one, look for more
    - repetition 
        - **"it*"** means zero or more t
    - alternation
        - | means or
    - character classes
        - **"[123]"** mean match anything in the set
    - excluding a character
        - **"[^a-z123]"** ^ indicates NOT any characters in this set
